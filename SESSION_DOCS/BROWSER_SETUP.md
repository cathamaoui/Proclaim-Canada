# Browser Integration Setup Guide

## Enable VS Code Browser Tools

To allow the chat to interact with your browser and app, you need to enable browser tools in VS Code settings.

### Step 1: Open Settings
1. Press `Ctrl + ,` (or `Cmd + ,` on Mac)
2. This opens VS Code Settings

### Step 2: Search for Browser Setting
1. In the search box, type: `workbench.browser.enableChatTools`
2. You should see the setting appear

### Step 3: Enable the Setting
1. Check the checkbox to **enable** `workbench.browser.enableChatTools`
2. This allows the chat interface to control the browser

### Step 4: Confirm
Once enabled, you'll see a popup asking to accept/allow browser tool usage. Click **Accept** or **Allow**.
